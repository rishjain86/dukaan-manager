## Quick deploy
1. Replace firebaseConfig in your dashboard/index files with your project's config.
2. Upload the entire folder to GitHub or Netlify (drag-n-drop).
3. Open `auto-setup.html`, paste firebaseConfig, run "Run Auto Setup" (enter owner UID and shopId).
4. Open `seed.html` on the same page (with firebase initialized) and run the seed script from the console or use the provided UI.
5. Test login / inventory / orders / coupons.

If anything breaks, capture console screenshot + failing network request and share.
